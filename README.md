# gamegenie

NES Game Genie encoder/decoder

## Installation

```
make && sudo make install
```

## Usage

```
Usage: gamegenie [-e,--encode <address> <data> (<compare>)]
                 [-d,--decode <code>]
                 [-v,--verbose]
```

## Explanation

Read more about how Game Genie codes work [here](http://tuxnes.sourceforge.net/gamegenie.html).
